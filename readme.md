# Portal Educativo - Aprende Desarrollo Web

## Descripción del Proyecto

Portal educativo oficial diseñado para que estudiantes dominen las habilidades del desarrollo web moderno. Esta plataforma ofrece una experiencia de aprendizaje completa y estructurada con cursos sobre tecnologías fundamentales del frontend.

## Características Principales

- **Navegación Intuitiva**: Menú principal para acceder a inicio, cursos, recursos y contacto
- **Diseño Responsivo**: Interfaz adaptable a cualquier dispositivo (desktop, tablet, móvil)
- **Cursos Estructurados**: 
  - HTML & CSS - Fundamentos del desarrollo web frontend
  - JavaScript - Programación interactiva en el navegador
  - Responsive Design - Sitios web para cualquier pantalla

- **Optimización Web**: Meta tags, preconexiones y precarga de recursos para mejor rendimiento

## Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos y variables CSS
- **Accesibilidad** - Soporta navegación ARIA y preferencias de movimiento reducido

## Cómo Usar

1. Abre el archivo `caso5.html` en tu navegador
2. Navega por las diferentes secciones usando el menú superior
3. Explora los cursos disponibles
4. Haz clic en "Comenzar ahora" para iniciar tu aprendizaje
